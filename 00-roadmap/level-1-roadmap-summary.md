# Level 1 Roadmap Summary

## Module 1 — Programming

Main outcomes:

- Understand Python basics
- Use virtual environments
- Use Git
- Understand OOP
- Build a FastAPI backend
- Connect to PostgreSQL
- Add JWT authentication
- Deploy with Nginx, systemd, domain, and SSL

## Module 2 — Operating Systems and Administration

Main outcomes:

- Use Linux confidently
- Harden SSH
- Configure firewall rules
- Create failed-login notifications
- Deploy self-hosted services
- Understand Windows administration basics
- Practice PowerShell, RDP, SMB/NTFS, Active Directory, and Group Policy

## Module 3 — Networking, Infrastructure and DevOps

Main outcomes:

- Understand DNS, DHCP, ARP, routing, NAT, firewalling, common ports
- Manage Mikrotik routers
- Build an ISP-style lab
- Build an enterprise network lab
- Configure routing/firewall rules between departments
- Practice VPN concepts

## Module 4 — OpSec

Main outcomes:

- Understand basic personal and technical OpSec
- Study real-world OpSec failure cases
- Understand encryption basics
- Learn public/private keys, AES, signatures, hashes, HTTPS, and metadata risks
